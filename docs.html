Overview

Language.fi is a character oracle and sentence staking protocol. It prices letters, numbers, spaces, and symbols using live usage data from token names, NFT collections, domains, registry entries, and exchange listings. Users mint or stake sentences whose values are calculated from the live prices of their underlying characters.

Core Invention

Characters become priced API primitives, and sentences become staked baskets of those primitives.

Meaning:

  • A, B, C, 7, SPACE, etc. each have live oracle prices
  • A sentence is valued by summing its characters
  • A staked sentence earns score based on its character basket, time held, diversity, and anti-spam rules

Value Logic:

More used character → higher demand score → higher primitive price → higher sentence value → stronger staking score or minting cost

Primitives

The protocol tracks the following primitives:

  • Letters: A-Z
  • Numbers: 0-9
  • Special primitive: SPACE
  • Optional symbols: ., !, ?, -, _, @, #

Primitive Data

Each primitive includes:

  • Current price
  • 24h change
  • Weekly change
  • Usage count
  • Source breakdown
  • Volatility
  • Rank
  • Staking weight
  • Last oracle update

Oracle

The Language.fi Oracle measures live letter and number usage across multiple data sources:

Oracle Sources

  • Solana token names (25% weight)
  • Solana NFT collections (20% weight)
  • Solana domains (15% weight)
  • Language.fi registry entries (25% weight)
  • Gate.io token listings (15% weight)

Live Data Integration

The oracle connects to live APIs:

  • CoinMarketCap Pro API - Token listings and names
  • Gate.io API - Trading pairs and symbols

Oracle Confidence

Oracle confidence is calculated based on sample size and source diversity to ensure reliable pricing data.

Sentence Staking

Users can stake sentences as character baskets that earn score based on:

Staking Score Formula

Staking Score = Sentence Value × Character Performance × Stillness Multiplier × Diversity Multiplier × Anti-Spam Score

Stillness Bonus

Longer-held sentences earn higher multipliers:

  • 0-7 days: 1.00x
  • 7-30 days: 1.10x
  • 30-90 days: 1.25x
  • 90-180 days: 1.50x
  • 180-365 days: 2.00x
  • 365+ days: 3.00x

Anti-Spam Rules

  • Penalties for repeated characters
  • Low diversity penalties
  • Length caps
  • Transfer resets stillness

API Reference

GET /api/primitives

Returns all primitives (letters, numbers, spaces, symbols)

GET /api/primitives/{symbol}

Returns detailed information for a single primitive

POST /api/sentences/quote

Get sentence pricing quote with character breakdown

POST /api/staking/sentence-score

Calculate staking score for a sentence

POST /api/oracle/update

Trigger live oracle data update

GET /api/oracle/live-stats

Get live oracle statistics from real token data

GET /api/primitives/live

Get primitives with live oracle data

POST /api/tokenize/oracle-stats

Tokenize oracle statistics as on-chain assets

GET /api/oracle/snapshot

Get oracle snapshot for settlement proof

Deployment

Prerequisites

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Edit .env with your API keys

Local Development

# Start development server
npm run dev

# Access at http://localhost:3000

Quick Deploy (Vercel)

# Deploy to Vercel
vercel --prod

Environment Variables

  • GATE_API_KEY - Gate.io API key
  • GATE_API_SECRET - Gate.io API secret
  • COINMARKETCAP_API_KEY - CoinMarketCap Pro API key
