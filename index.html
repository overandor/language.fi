Explainable letter pricing.

The oracle uses multiple source categories to determine letter prices with full transparency.

Registry Activity

Counts letters used in minted Language.fi assets.

Token Name Data

Counts letters from sampled token names and market-native naming behavior.

Blockchain Hash Data

Uses sampled hashes and addresses as a cryptographic frequency baseline.

Text Sample Data

Uses permitted or licensed text samples, public metadata, RSS feeds, or derived frequency counts.

Seasonal and Meme Trends

Captures temporary demand spikes from cultural naming patterns.

Pricing Formula
Letter Price = Base Price + Registry Demand + Token Name Demand + Hash Baseline + Text Sample Demand + Seasonal Modifier + Popularity Tax - Decay Adjustment
Important: Language.fi stores derived counts and statistics from third-party text sources, not full copyrighted articles.
